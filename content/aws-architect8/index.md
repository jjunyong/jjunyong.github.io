## IaC
IaC의 장점 : 재사용성과 업데이트

### AWS CloudFormation
CloudFormation > Stack > create Stack에서 IaC yaml파일을 등록하여 한 꺼번에 프로비저닝 할 수 있다.
- AWS 대부분의 리소스를 CloudFormation 템플릿으로 제어할 수 있다. 

### AWS Elastic Beanstalk 
웹 콘솔에서 클릭으로 할 수 있기 때문에 편의성 면에서 프로비저닝하기 더 좋다
